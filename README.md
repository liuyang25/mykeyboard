自用键盘映射工具
# 使用方法：
* windows   
直接执行keyboardRemapper.exe   
基于autohotkey+脚本，安装autohotkey后可通过修改脚本keyboardRemapper.ahk 重新生成exe

* mac
基于karabiner
导入自定义模板ly.json

# 功能
使用caplock开关，实现以下映射：
k::Up  
h::Left  
j::Down  
l::Right  
u::PgUp  
n::PgDn  
i::Home  
m::End  
o::Del  
p::Backspace  

# 注
windows为开关方式  
mac为组合方式
